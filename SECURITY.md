# Security Policy

## Reporting a Vulnerability
Please report security issues privately.

Preferred:
- GitHub private vulnerability reporting for this repository, if enabled.

Fallback:
- Email: rajugowda.towork@gmail.com

Please include the affected version, impact, and repro steps. Do not open public issues for active vulnerabilities.

## Supported Versions
| Version | Supported |
| --- | --- |
| 1.1.x | Yes |
| 1.0.x and earlier | No |

## Response Expectations
We aim to acknowledge security reports within 3 business days and provide a follow-up plan after triage.
